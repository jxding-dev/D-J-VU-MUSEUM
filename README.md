# 기시감 박물관

꿈, 기억, 이미지, 문장 조각이 잘못 보관된 가상 박물관을 다루는 인터랙티브 에디토리얼 웹사이트입니다.

## 실행

`run.bat`을 더블클릭하거나 아래 명령을 실행합니다.

```bash
npm install
npm run dev
```

로컬 주소:

```text
http://127.0.0.1:5173/
```

## 사용 기술

- React + Vite
- GSAP + ScrollTrigger
- Motion
- React Three Fiber
- CSS 변수, `clamp()`, `backdrop-filter`, `mix-blend-mode`, `clip-path`
